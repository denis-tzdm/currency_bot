# Telegram-бот для конвертации валют

Бот получает актуальные данные через
[Exchange Rates Data API (APILayer.com)](https://apilayer.com/marketplace/exchangerates_data-api),
кэширует полученные данные на указанный в конфигурации период актуальности в базе [redis](https://redis.com/).

Практическая работа по блоку С5.6 (ООП) курса SkillFactory "Интенсив «Python-разработчик»".
Поток INTPY-6.
